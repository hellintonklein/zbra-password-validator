# ZBRA Challenge

- As instruções para o teste estão dentro do diretório `instructions`;
- Indicamos que use React, Angular ou TypeScript (fique à vontade);
- Você terá 3 dias para entregar o projeto (tudo bem se levar um pouco mais de tempo);
- Consideraremos apenas código presente no branch `main`;
- Com este projeto, queremos entender seus conhecimentos!

E por fim, esperamos que você entregue o seu melhor!

## Uso de IA / LLMs

O uso de ferramentas de inteligência artificial (como GitHub Copilot, ChatGPT, Claude, entre outras) é **opcional** neste desafio. O resultado será avaliado independentemente do uso ou não de IA. Porém, caso seja utilizado, espera-se que o candidato:

- **Compreenda** todo o código presente na solução entregue, independentemente de como foi gerado;
- **Seja capaz de explicar** as decisões de arquitetura, padrões utilizados e o funcionamento geral da aplicação durante uma eventual revisão técnica;
- **Não substitua o raciocínio próprio** pelo da IA: use as ferramentas como apoio, não como substituto do seu entendimento;
- **Informe** no README da sua solução se e como utilizou ferramentas de IA durante o desenvolvimento.

O objetivo do desafio é avaliar suas habilidades e conhecimentos. Entregas onde o candidato não consegue explicar o próprio código serão desclassificadas.

Caso tenha utilizado ferramentas de IA, esperamos que o candidato inclua no repositório (branch `main`) os seguintes arquivos, quando aplicável:

- **`CLAUDE.md`** (ou equivalente para a ferramenta utilizada): arquivo de configuração/instruções fornecidas ao modelo;
- **`PLAN.md`**: planejamento ou roteiro gerado pela IA antes da implementação;
- Quaisquer outros artefatos relevantes que descrevam as **skills, prompts ou estratégias** utilizadas durante o desenvolvimento.

Esses arquivos nos ajudam a entender melhor como a IA foi utilizada no processo e fazem parte da avaliação.
